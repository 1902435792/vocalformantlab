# Vocal Formant Lab

**极简启动指南**

想要运行这个项目？只需简单的 3 步：

## 第一步：安装基础环境
电脑上必须有 **Node.js** 才能运行。
- **去哪里下**：[官网下载链接 (长期支持版)](https://nodejs.org/zh-cn/download/prebuilt-installer)
- **怎么安装**：下载后双击安装包，一路点 Next (下一步) 直到完成。

## 第二步：下载本项目
- 点击页面绿色的 **Code** 按钮 -> **Download ZIP**。
- 下载后解压到电脑任意位置。

## 第三步：一键启动
1. 进入解压后的文件夹。
2. 找到 **`启动应用.bat`** 文件，双击它。
3. **喝杯水等着**：脚本会自动帮你下载依赖、配置环境、最后打开浏览器。

---
*首次运行如果提示缺Key，脚本会自动暂停并帮您生成配置文件，您只需填入即可。*
